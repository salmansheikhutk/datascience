Breast Cancer Analysis

Machine learning classification using logistic regression

Dataset Overview

The dataset originates from the Breast Cancer Wisconsin (Diagnostic) Data Set, a widely used benchmark in machine learning for medical diagnostics. It contains detailed measurements of cell nuclei from breast tissue samples, enabling the classification of tumors as either benign (non-cancerous) or malignant (cancerous).

Source: University of Wisconsin, provided through Kaggle

Purpose: Binary classification to predict tumor type (benign or malignant)

Size: 569 samples with 30 features measuring cell characteristics

Methodology

1. Data Loading

Imported dataset from Kaggle with 569 samples and 32 columns

2. Data Cleaning

Removed unnecessary columns (ID, unnamed) and converted M/B labels to 1/0

3. Feature Selection

Split data into features (X) - 30 measurements, and target (y) - diagnosis

4. Train-Test Split

Divided data: 80% training (455 samples), 20% testing (114 samples)

5. Model Training

Trained Logistic Regression model using scikit-learn

6. Evaluation

Assessed performance using accuracy, precision, recall, and F1-score

Key Features

Features: 30 numerical measurements including radius, texture, perimeter, area, smoothness, compactness, concavity, symmetry, and fractal dimension

Target Classes: Benign (357 samples, 62.7%) vs Malignant (212 samples, 37.3%)

Algorithm: Logistic Regression with random_state=42 for reproducibility

Validation: Comprehensive evaluation using confusion matrix and multiple performance metrics
